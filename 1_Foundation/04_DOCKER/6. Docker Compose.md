## ğŸ”¹ Running Docker Compose
- Start all services:
  ```bash
  docker-compose up
  ```
- Run in background (detached mode):
  ```bash
  docker-compose up -d
  ```
- Stop services:
  ```bash
  docker-compose down
  ```
- Rebuild containers:
  ```bash
  docker-compose build
  ```
