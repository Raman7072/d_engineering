## 4. Writing a Dockerfile (Example for a Python Flask App)
```Dockerfile
# base image
FROM python:3.11

# work directory
WORKDIR /app

# copy
COPY . .

# run
RUN pip install -r requirements.txt

# port
EXPOSE 5000

#command
CMD ["python", "app.py"]
```
#### Breakdown:
- `FROM python:3.11` â†’ Use Python 3.11 as the base image.
- `WORKDIR /app` â†’ Set working directory inside container.
- `COPY . .` â†’ Copy project files from local machine to container.
- `RUN pip install -r requirements.txt` â†’ Install dependencies during image build.
- `EXPOSE 5000` â†’ Inform Docker that the app uses port `5000`.
- `CMD ["python", "app.py"]` â†’ Run this command when container starts.

## 5. Building & Running Your App
```bash
# Build the image
docker build -t my-python-app .

# Run the container and map port 5000
docker run -p 5000:5000 my-python-app
```

## 7. Useful Docker Commands
- List Images
  ```bash
  docker images
  ```
- Check Running Containers
  ```bash
  docker ps
  ```
- View Logs
  ```bash
  docker logs <container_id>
  ```
- Access Inside a Container
  ```bash
  docker exec -it <container_id> /bin/bash
  ```
- Stop a Container
  ```bash
  docker stop <container_id>
  ```
- Remove an Image
  ```bash
  docker rmi <image_id>
  ```
